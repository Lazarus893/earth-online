# Earth Online - 素材生产手册

> 原则：同组素材共享相同的基础风格描述，批量生成时保持视觉一致性。
> 每组开头有【组风格基底】，每个素材的 prompt 已内含该基底，可直接复制使用。
> 视频素材直接标注了打底图依赖。
