# Game Design Document - Earth Online

## Core Fantasy
你是一个在"地球Online"这个 MMORPG 中的玩家。现实生活就是游戏本体。
AI 是你的任务向导和技能导师，帮你规划升级路线并在 Actions 层"施放技能"。

## Dimension System (维度系统)

### Physical (体力) - 初始解锁
**代表**: 身体健康、运动能力、睡眠质量、饮食
**Score 影响因素**:
- 运动频率和强度
- 睡眠时长和质量
- 饮食规律性
- 身体指标变化

**Goals 示例**:
- 增肌/减脂到目标体重
- 建立规律运动习惯
- 改善睡眠质量

### Energy (精力) - 初始解锁
**代表**: 精神状态、专注力、情绪管理、心理健康
**Score 影响因素**:
- 每日精力自评
- 专注工作时长
- 情绪波动频率
- 冥想/放松习惯

**Goals 示例**:
- 建立冥想习惯
- 提升深度工作能力
- 建立情绪管理框架

### Career (职业) - 初始解锁
**代表**: 专业技能、工作产出、学习能力、行业地位
**Score 影响因素**:
- 技能学习进度
- 项目完成质量
- 行业影响力
- 收入增长

**Goals 示例**:
- 90天内掌握AI内容创作
- 完成某个技术认证
- 建立个人品牌

### Social (社交) - 锁定
**解锁条件**: Energy >= Lv5 AND Career >= Lv3
**代表**: 人际关系、社交能力、影响力、协作能力
**设计理由**: 良好的精力状态和一定的职业能力是有效社交的基础

### Finance (金钱) - 锁定
**解锁条件**: Career >= Lv5 AND Physical >= Lv3
**设计理由**: 稳定的职业收入和良好的身体基础是理财的前提

## Progression Mechanics (进阶机制)

### Experience Points (经验值)
```
Final EXP = base_value × difficulty_mult × streak_mult × patch_mult × combo_mult

difficulty_mult: trivial(0.5) easy(1.0) medium(2.5) hard(5.0) epic(10.0)
streak_mult: 连续天数对应的加成
patch_mult: 世界状态加成/减益
combo_mult: 当天完成多个同维度任务的连击加成 (1.0, 1.1, 1.2, ...)
```

### Level Curve (等级曲线)
```
Level N 所需总 EXP = base × growth_rate^(N-1)
Level 1: 100 EXP
Level 5: 174 EXP  
Level 10: 304 EXP
Level 20: 929 EXP
Level 50: 15,415 EXP
```

### Decay System (衰减系统)
现实中不使用就会退步，游戏也是：
- 每天未活跃的维度 -1% score
- 2天宽限期（不扣分）
- 最低不低于 10 分（保底）
- 衰减提醒：系统会在衰减前预警

## World Patch System (版本更新)

### Patch Types
| 类型 | 频率 | 影响范围 | 示例 |
|------|------|----------|------|
| Hotfix | 即时 | 单维度 | 突发暴雨取消户外计划 |
| Daily Patch | 每日 | 1-2维度 | 天气变化、新闻热点 |
| Weekly Patch | 每周 | 全维度 | 经济数据、行业趋势 |
| Season Update | 每季 | 全局 | 季节变换、大版本更新 |

### Patch Note Format
模拟游戏更新公告，让用户感受"世界在变化"：
```
═══════════════════════════════════════
  EARTH ONLINE - Patch v2026.05.12
═══════════════════════════════════════

[WEATHER MODULE]
• 上海: 晴转多云 26°C
• 户外活动经验值 +15%
• 建议: 下午3点前完成户外任务

[CAREER MODULE]  
• AI行业动态: 新模型发布
• 新技能分支可用: "多模态内容创作"
• 学习任务权重 +10%

[ECONOMY MODULE]

═══════════════════════════════════════
```

## Skill Tree (技能树)

AI 在 Actions 层的能力随等级解锁：

### Tier 1: Basic (Lv 1-3)
- 信息搜索与摘要
- 基础日报生成
- 简单任务提醒
- 基础计划模板

### Tier 2: Advanced (Lv 4-7)
- 个性化分析报告
- 多源信息整合
- 自动化工作流
- Notion 知识库管理
- 设计素材生成

### Tier 3: Expert (Lv 8-10)
- 预测性建议（基于历史数据）
- 跨维度联动优化
- 创造性内容产出
- 复杂项目编排
- 个人品牌内容策略

## Interaction Philosophy (交互哲学)

### 原则
1. **一键优先**: 能点就不打字
2. **智能推荐**: AI 提供 2-4 个选项 + 自定义
3. **即时反馈**: 每个操作都有视觉/数值反馈
4. **仪式感**: 重要节点有特殊动画和提示
5. **低压力**: 不强制、不惩罚过重、鼓励为主

### 交互模式
```
常规任务确认:
  [✓ 完成] [→ 跳过] [↻ 延期] [✏ 修改]

每日开始:
  "今天的冒险准备好了吗？"
  [⚔ 开始今日任务] [📋 查看详情] [🔄 调整计划]

状态更新:
  "感觉今天精力如何？"
  [🔥 充沛] [😊 不错] [😐 一般] [😴 疲惫]
```
